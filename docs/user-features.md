# BetterToLive 功能文档

BetterToLive 是一个把日常记录、内在整理、关系梳理和未来规划放在同一工作台里的个人生活系统。

## 你可以这样用

- 用顶部搜索在当前页面快速筛选内容。
- 用右上角扳手切换到控制模式，在支持的页面新增、编辑、删除内容。
- 用右上角工具区切换主题、查看通知、播放背景音乐、切换中英文界面。
- 用“快速记录”直接进入反思页，把当下先写下来。

## 功能一览

- 总览：查看最近记录、今天的观察和常用入口。
- 反思：写反思、看记录库、整理主题脉络和回看问题。
- 记事：记录生活事件，按时间线和主题回顾。
- 记账：记录收入支出，查看分布，维护预算规则和复盘线索。
- 购物：管理物件库、物件系统、空间场景和阶段适用。
- 饮食：管理每日计划、食谱、食品分类、营养成分表和进食记录。
- 情绪情感：记录今天的情绪，回看波动时间线、触发因素和恢复工具。
- 观念：整理观念清单、三层骨架、相互关系和盲区问题。
- 原则：整理原则、边界、支撑/冲突关系和实践回路。
- 关系深化：维护人物档案、想说的话、关键互动和跨关系模式。
- 成长记忆：整理记忆时间线、成长节点、锚点、回看问题和影响线索。
- 生命整理：整理重要托付、交付地图、关系表达和信任边界。
- 社会经济：整理外部经济认知、关联图谱、认知缺口、复习提问和决策距离。
- 未来：编写理想自我与生活蓝图、阶段路径、当前实验和对齐检查。

## 当前工作台结构

- 原则：总览、个人原则、他人原则、校准实践。
- 社会经济：总览、经济学、社会学、知识关联、学习队列。

## 本轮结构改造记录

### 需求分析与设计

- 目标：把单页模块逐步改造成类似购物模块的页内多 Tab 工作台，而不是再套一层页面级切换。
- 原则模块设计：保留总览用于分类、关系与边界总览，同时把“个人原则 / 他人原则 / 校准实践”拆成独立入口，方便区分哪些原则真正属于自己。
- 社会经济模块设计：在总览之外拆出“经济学 / 社会学 / 知识关联 / 学习队列”，让学科视角、概念关系与复习队列各自承担清晰职责。
- 内容策略：优先复用现有业务字段与结构，在不破坏编辑流的前提下补充学科、视角等显式标签，并为旧数据提供推断兼容。
