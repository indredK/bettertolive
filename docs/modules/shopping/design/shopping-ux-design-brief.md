# bettertolive 购物 · 功能说明

目标：购物让生活更美好，不是买得更多。不打分、不心理量化；只靠场景、阶段、空间和冷静。

---

## 1. 场景

- 按生活场景看家里各处：厨房、桌面、睡眠、活动、清洁、待客、玄关、通勤等
- 每个场景三种状态（白话）：缺一点 / 有点满 / 还行；也可用「用不到」
- 点开场景：看相关物品、还缺什么、可以先收什么
- 想买时先选场景归类，再决定要不要买

## 2. 阶段

- 记录当前生活阶段（如：独居 · 租房 · 远程）
- 对某件东西做对照：如果变成合住 / 搬家 / 回办公室 / 长期出门 / 有娃…
- 给出白话结论（合适 / 占地方 / 可以再等等等），不评分
- 结论后可：放进冷静室、以后再说、不要了

## 3. 冷静室

- 想买的东西可先放几天（默认约 72 小时，可再延）
- 条目显示还剩多久
- 到期或随时可操作：还想要 / 再放几天 / 算了
- 「算了」是正常结果，不是失败
- 全局可看到冷静中件数

## 4. 空间

- 按位置看放不放得下：桌面、厨房台面、橱柜、客厅、玄关、衣柜等
- 状态：松 / 满 / 挤
- 想买大件前可先看对应位置；满了可先收或先冷静

## 5. 物品

- 物品可关联：场景、摆放位置、阶段
- 状态包括：冷静中、想买、已有、以后再说
- 可标记已有；可从场景入口进入「看看 → 冷静 / 换阶段 / 已有」

## 6. 与现有模块的对应（功能层）

| 能力 | 大致落在 |
|------|----------|
| 场景 | Systems / Overview |
| 空间 | Spaces |
| 阶段 | Stages |
| 物品库与状态 | Planning / Item Library |
| 冷静室 | 新增全局能力（角标 + 列表） |

---

## 7. 不做

- 生活分、负荷指数、幸福量化
- 购前问卷 / 必要性量表 / 节奏标签堆砌
- 推荐爆款、促销逼单
