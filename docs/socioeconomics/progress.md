# 社会经济模块进度文档

## 已完成

- 已按 `design.md` 的 7 个领域、3 个层次、4 级掌握程度、5 种来源、4 级决策距离落地社会经济模块的数据结构与页面展示。
- 已新增 Tauri 本地后端模块 `src-tauri/src/socioeconomics/`，通过 `get_socioeconomics / save_socioeconomics` 提供社会经济模块的读取与保存能力。
- 已提供一批基于设计文档编排的后端 seed 数据，覆盖 `entries / gaps / reviewPrompts`，用于首次启动时初始化社会经济模块。
- 已移除前端独立的社会经济 mock 数据文件，前端 live 链路改为读取本地后端数据；mock 模式下仅保留空兜底结构。
- 已为社会经济页面补齐响应式一页展示：宽屏使用固定仪表盘布局，窄屏切换为堆叠布局，保持相同信息架构。
- 已参考购物模块补齐控制模式：在管理模式下支持新增、编辑、删除认知条目、认知缺口和复习提问。
- 已处理搜索过滤与保存的边界：展示使用过滤结果，保存使用原始未过滤模块数据，避免误删隐藏记录。
- 第二轮严格复查已修复复习提问的索引映射：搜索过滤后点击编辑会先映射回完整源数组 index，再保存或删除，避免字符串列表因显示 index 偏移而改错隐藏提问。
- 已补齐社会经济模块的国际化文案与枚举翻译，并为三套主题新增社会经济专用语义色变量，供热力图、优先级标签和分布条使用。
